Envio Automático (Fonnte)

Como configurar (gratuito)

  1. Acesse fonnte.com e crie uma conta gratuita
  2. Vá em Devices → Add Device
  3. Escaneie o QR Code com o WhatsApp do seu celular (igual ao WhatsApp Web) — faça isso uma única vez
  4. Com o device conectado, clique nele e copie o Token que aparece no topo
  5. Cole o token no campo abaixo, ative o toggle e salve

Gratuito até 1.000 mensagens/mês. Após configurado, toda vez que salvar um agendamento a mensagem é enviada automaticamente.
